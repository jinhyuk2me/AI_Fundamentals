## TL;DR
- 측도론은 확률을 엄밀히 정의하고, 일반화·수렴성·Optimal Transport 이론의 토대가 되는 추상적 적분 체계다.
- 이 노트는 Phase 4 측정/확률 심화 작업을 위한 템플릿으로, [[Math/Math_확장_계획]]의 완료 기준(정의→예제→연습문제→링크)을 미리 반영한다.
- 르베스그 적분과 시그마-대수를 명확히 도입해 향후 확률 과정/최적 운송 노트와 상호 연결될 예정이다.

## 언제 쓰나
- 일반화 이론 연구에서 수렴 개념(거의 확실한 수렴, $L^p$ 수렴 등)을 증명할 때
- Variational Inference, Diffusion, Optimal Transport에서 측도 기반 비용 함수를 다룰 때
- 확률 과정, 마르코프 체인, 스토캐스틱 미분방정식의 엄밀한 정의가 필요할 때

## 주요 API
| 이름 | 설명 | 예정 실습 |
| --- | --- | --- |
| $\sigma$-대수 | 집합 계량 구조, 확률 측도의 정의역 | Python으로 간단한 $\sigma$-대수 생성 확인 |
| 르베스그 적분 | 측도에 대한 적분, 확률 변수 기댓값의 일반형 | Riemann vs. Lebesgue 비교 실험 |
| 확률 측도 $\mathbb{P}$ | $(\Omega, \mathcal{F})$ 위에 정의된 측도 | 샘플 공간 시뮬레이션으로 규칙 검증 |
| 지배 수렴 정리 | 수렴과 적분 순서를 교환하는 정리 | Monte Carlo 예제에서 실증 |

## 실습 예제
1. 단순 함수 근사를 이용해 르베스그 적분을 수치적으로 구현하고, Riemann 적분과의 오차를 그래프로 비교 (작성 예정).
2. Python으로 유한 $\sigma$-대수를 생성해 폐쇄성 규칙을 검증하고, 확률 측도의 정의를 실험적으로 확인 (작성 예정).
3. Dominated Convergence Theorem을 Monte Carlo 샘플로 확인해 Variational Inference에서의 교환 조건을 설명 (작성 예정).

## 실수 주의
- $\sigma$-대수 기호와 집합 연산을 혼동하면 증명 스케치가 무너질 수 있으므로 기호 표를 따로 관리한다.
- 르베스그 적분의 "정의역"과 "적분 대상 함수"를 혼동하면 측도 공간 설정이 부정확해진다.
- 확률 측도 업데이트 시 조건부 측도를 명시하지 않으면 이후 [[Math/Probability/5. 베이지안 추론]]과 연결이 어렵다.
